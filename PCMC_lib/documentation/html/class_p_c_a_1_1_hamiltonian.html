Detailed Description

Hamiltonian.

\[ H=-\sum_{i=1}^{N-1} (2+\mu)\kappa_{i+1} \kappa_i + \alpha\sum_{i=1}^{N} \left\{ 2\kappa_i ^2 + q ( \kappa_i^2-m^2)^2+ \frac{c}{2}(d \kappa_i^2+1) \tau^2-a(b\kappa_i^2+1)\tau_i\right\} \]

§ generateTau()

double PCA::Hamiltonian::generateTau ( double  kappa_i,
double  T

Generate tau according Gaussian distribution.

\[P\sim \exp\left(-\frac{(\tau_i-\mu)^2}{2\sigma^2}\right)\]

Coefficients:

\[\mu=\frac{a(b\kappa_i^2+1)}{c(d\kappa_i^2+1)}\]

\[\sigma^2=\frac{T}{\alpha c(d\kappa_i^2+1)}\]
